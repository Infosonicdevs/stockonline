# Sachiv Mitra Sale Client

This is the frontend application for the Sachiv Mitra Sale system. It is designed to manage customers, ledgers, stock, and transactions for a society/outlet management system.

## 🚀 Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 7](https://vitejs.dev/)
- **Styling**: 
  - [Tailwind CSS 4](https://tailwindcss.com/)
  - [Bootstrap 5](https://getbootstrap.com/)
  - [Bootstrap Icons](https://icons.getbootstrap.com/)
- **Routing**: [React Router Dom 7](https://reactrouter.com/)
- **HTTP Client**: [Axios](https://axios-http.com/)
- **Notifications**: [React Toastify](https://fkhadra.github.io/react-toastify/)

## 📂 Project Structure

- `src/pages`: Contains all the page components.
- `src/services`: Contains API service files for backend communication.
- `src/shared`: Generic and reusable components.
- `src/utils`: Helper functions and utilities.
- `Documents`: Project documentation (API, instructions).

## ✨ Key Features & Pages

- **Dashboard**: Overview of system statistics.
- **Customer Management**: Registering and managing customer information and lists.
- **Ledger Management**: Handling ledgers, groups, sub-groups, and balances.
- **Stock/Inventory Management**: Managing stock groups, sub-groups, balances, and details.
- **Supplier Management**: Information and balance management for suppliers.
- **Voucher Entries**: Recording financial transactions.
- **Location Management**: Settings for States, Districts, Talukas, and Cities.
- **System Settings**: User management, roles, prefixes, society/outlet info, and financial year settings.
- **GST Management**: Configuration of GST slabs.

## 🛠️ Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Set up environment variables in `.env`.
4. Run in development mode: `npm run dev`
